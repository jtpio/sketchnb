# sketchnb

A collection of generative art sketches as Jupyter notebooks, powered by the [p5.js kernel for JupyterLite](https://github.com/jupyterlite/p5-kernel).

## Sketches

- **sketch.ipynb** — A simple interactive p5.js demo
- **abyssal-bloom.ipynb** — Bioluminescent flow field driven by Perlin noise
- **painting-with-data.ipynb** — Climate spiral visualization using NASA GISS data and imported npm packages (chroma-js)

## Agent Skill

This repo includes an [agent skill](https://agentskills.io/home) in `_agents/skills/algorithmic-art/` for generating algorithmic art notebooks. It is loosely based on the [algorithmic-art skill](https://github.com/anthropics/skills/blob/main/skills/algorithmic-art/SKILL.md) from the Anthropic skills collection.
